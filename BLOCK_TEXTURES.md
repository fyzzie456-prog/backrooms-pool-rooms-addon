- `pool_water_special.png` - Pale cyan water with subtle ripples
- `water_drain.png` - Dark gray circular drain grate

### FURNITURE & FIXTURES
- `bench.png` - Light gray metal bench seat
- `locker.png` - Silver/gray metal locker with handle details
- `metal_shelf.png` - Industrial metal shelving texture
- `pool_ladder.png` - Stainless steel ladder with grip

### WALLS & DECOR
- `wall_pipe.png` - Gray metal piping texture
- `wall_vent.png` - Square ventilation grate pattern

- `mirror_wall.png` - Reflective white mirror surface

## Texture Specifications

- **Resolution**: 16x16 or 32x32 pixels
- **Format**: PNG with transparency support
- **Color Space**: RGB or RGBA
- **Compression**: Standard PNG compression

## Design Tips for Backrooms Aesthetic

1. **Keep it Clean**: Use smooth, simple designs
2. **Subtle Patterns**: Add light textures for depth
3. **Realistic Materials**:
   - Tiles: Slight grout lines, smooth finish
   - Metal: Subtle metallic sheen, maybe some wear
   - Glass: Slight reflection effect
   - Concrete: Fine grain texture

4. **Color Palette**:
   - Whites: #FFFFFF, #F5F5F5, #F0F0F0
   - Grays: #C0C0C0, #A9A9A9, #808080
   - Cyans: #4DD0FF, #8EFFFF, #E0FFFF
   - Accent: #FFD700 (yellow for caution)

5. **Imperfections**: Add slight variations to avoid perfect uniformity

## Creating Textures (Quick Guide)

### Using Aseprite, Photoshop, or GIMP:

1. Create new image 16x16 pixels
2. Set background to white
3. Add subtle details (grout lines, wear, etc.)
4. Export as PNG
5. Place in resource_pack/textures/blocks/

### Using Minecraft Texture Tools:
- Use existing block textures as reference
- Modify vanilla textures for custom blocks
- Test in-game immediately

## Testing Your Textures

1. Save texture files to resource_pack/textures/blocks/
2. Enable resource pack in world settings
3. Enter the End dimension
4. Use commands to place blocks:
   ```mcfunction
   setblock ~ ~ ~ white_concrete
   ```
5. Check appearance and adjust colors/patterns as needed

## Performance Notes

- PNG format recommended for compatibility
- 16x16 textures are standard
- Larger textures (32x32+) work but use more memory
- Mipmaps automatically generated by Minecraft

## Alternative: Using Vanilla Textures

If you don't want to create custom textures, the addon works great with vanilla blocks:
- `white_concrete` for tiles
- `sea_lantern` for lights
- `light_gray_concrete` for walls
- `iron_block` for metal fixtures
